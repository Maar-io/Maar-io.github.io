Polkadot Github Explorer

I made a simple app Dotctor, to explore Polkadot related projects on the Github. It searches Github's topic combinations like substrate+wallet. You need to log-in to the Github for this app to work. It uses Github GraphQl to search for the given topic combination. The default is polkadot+blockchain. You can of course use Github to do topic search but I'm not aware you can do it for the combination of the topics. I did a bit of investigation and I think majority of the substrate based projects are captured. That been said, unfortunately, not all projects use topics to describe itself, and those projects will be omitted.

  • ReactJs with react-bootstrap
  • GraphQl with Apollo client
  • Firebase Authentication with Github as OAuth identity provider
  • Hosted by Firebase
